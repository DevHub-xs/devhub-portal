# Code of Conduct

## Professional Standards

As contributors and maintainers of this project, we pledge to foster a professional, respectful environment.

### Expected Behavior

- Professional communication
- Constructive feedback
- Respect for differing viewpoints
- Focus on project goals
- Collaborative problem-solving

### Unacceptable Behavior

- Harassment or discrimination
- Unprofessional conduct
- Spam or trolling
- Disclosure of private information
- Any illegal activities

## Enforcement

Violations may be reported to: **paccarini.bar@outlook.com**

Inappropriate behavior may result in:
- Warning
- Temporary restriction
- Permanent ban from project

## Scope

This Code of Conduct applies to all project spaces and public representation of the project.

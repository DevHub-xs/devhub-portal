# Contributing Guidelines

## Proprietary Software Notice

This is proprietary software owned by Pedro Accarini. Contributions are only accepted from authorized contributors.

## Development Setup

1. **Prerequisites**
   - Node.js >= 18
   - npm >= 9
   - Angular CLI 17+

2. **Installation**
   ```bash
   npm install
   ```

3. **Development Server**
   ```bash
   npm start
   ```

## Code Standards

### Angular/TypeScript
- Follow Angular Style Guide
- Use standalone components
- Implement OnPush change detection
- Write unit tests for components/services

### Styling
- Use Tailwind CSS utility classes
- Follow mobile-first approach
- Maintain consistent spacing/typography

### Git Workflow
- Branch naming: `feature/`, `bugfix/`, `hotfix/`
- Commit format: `type(scope): description`
  - Types: feat, fix, docs, style, refactor, test, chore

## Pull Request Process

1. Update documentation if needed
2. Ensure all tests pass
3. Follow code review feedback
4. Squash commits before merge
